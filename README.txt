Registro de Mantenimiento Diario - Mindray

Esta aplicación permite registrar actividades de mantenimiento realizadas al equipo MINDRAY, tales como:
- Eliminación de Desechos
- Limpieza Sonda de Muestra
- Ab Limp Electrica
- Desobs
- Baño WBC, RBC, DIFF
- Cámara de flujo
- Sonda Muestra

Incluye:
- Registro de fecha, hora y operador
- Respaldo automático
- Visualización mensual de registros
- Filtro y descarga por mes
